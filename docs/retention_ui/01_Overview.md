# Overview (UI)

UI supports operator workflow:

1. Paste/upload JSON dataset (REQ-0001/REQ-0002). [Source: docs/inputs/requirements_source.md#UI-REQ-0001-—-Paste-Dataset-JSON]
2. Client validation (REQ-0003) and server validation via API (REQ-0004). [Source: docs/inputs/requirements_source.md#UI-REQ-0004-—-Server-Side-Validation]
3. Set `releasesToKeep` (REQ-0005) then evaluate (REQ-0006). [Source: docs/inputs/requirements_source.md#UI-REQ-0006-—-Execute-Evaluation]
4. Inspect kept releases and decision log (REQ-0007/REQ-0008). [Source: docs/inputs/requirements_source.md#UI-REQ-0007-—-Display-Kept-Releases]
5. Export raw response (REQ-0009). [Source: docs/inputs/requirements_source.md#UI-REQ-0009-—-Export-Results]
